#### Requirements

* Python 3.13.2
* Flask 3.1.0
* flask-cors 5.0.1
* Jinja2 3.1.6
* Werkzeug 3.1.3
* python-dotenv 1.1.0
* requests 2.32.3
* click 8.1.8
* itsdangerous 2.2.0
* MarkupSafe 3.0.2


#### How to install

```sh
git clone https://github.com/PJY00/LLPC_RoutePilot.git
cd LLPC_RoutePilot
pip3 install -r requirements.txt
```
